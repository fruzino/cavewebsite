Source Code

Cave is open source. You can clone the repository and build it from source using a C compiler.

git clone https://github.com/limesuggestbox360/cave/cave.git
cd cave
gcc cave.C -o cave

Post-Installation

Need help installing? Check out our Step-by-Step Installation Guide with pictures.

After installing, you can verify the installation by running:

cave --version
