Installation Guide

Follow these simple steps to get Cave up and running on your machine.

Step-by-Step Instructions

Download page

Go to the official download page

Download version

Download the latest version.

Suscipison errors

If you get a suscipison error.

This is because the .exe file is not signed. Do not worry!

Download

CLick the download button.

Run the Setup

Run the setup file called "CaveSetupX_X_X" (may be different).

Finishing installations

Click "Finish" once downloaded.

Locating

The 'Cave' folder is usually in the 'Program Files x86' folder.

Verify Installation

Once clicked on the folder you will see this. Double click the cave.exe file.

Run anyway

If you get the suscipison error. Click 'Run anyway'.

This is also because cave.exe is not signed. In the future this error may be gone.
