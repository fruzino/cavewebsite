About Cave

Why use Cave?

Built on C, Cave is fast and easy to learn.

Syntax

Easy syntax best for all beginners starting out with C or Python.

Standard Usage

Write clean, expressive code with minimal boilerplate. Here is a sample of Cave in action:

// Define variables with type prefixes
!count = 5
@name = "Explorer"

// Loop with a clear condition
#while (count > 0) {
    write("Hello " + name + "! Counting: " + count)
    !count = count - 1
}

// Logic statements starting with #
#if (count == 0) {
    write("Done!")
}

Community & Development

Cave is an open ecosystem. Install community plugins with ease or build your own to extend the language's power.

$ cave cli install my-plugin
$ cave run script.cv
