Documentation & Syntax

Welcome to the Cave documentation. This guide covers the basic syntax and core commands of the language.

Variables and Types

Cave uses prefix characters to determine the type of a variable upon declaration.

Prefix Type Example
! Integer !count = 10
? Float ?pi = 3.14
@ String @msg = "Hello"
& Boolean &0 (false), &1 (true)

Control Flow

Logic and loops start with the # character.

Keyword Description Example
#if Conditional logic #if (x > y) { ... }
#while Conditional loop #while (x < 10) { ... }
#turn Repeat N times #turn (5) { ... }
#loop Infinite loop #loop { ... }
#for Range Loop #for (!i in r(10)) { ... }

Functions

Functions are declared with the function keyword and called with get.

function greet(name; msg) {
    write(msg + " " + name)
}

get greet("Explorer"; "Welcome")

File Operations

The file object provides several useful methods for file system interaction.

  • file.create(filename): Create a new file.
  • file.edit(filename, content): Write content to a file.
  • file.read(filename): Read content from a file.
  • file.delete(filename): Deletes a file.
  • file.run(filename): Executes a file's content as Cave script.

System & Console

Commands to interact with the environment and output.

Command Action Example
write(_) Clear Terminal write(_)
write(&color, text) Print Colored Text write(&red, "Error!")
scan(var, text) Prompt User Input scan(!age, "Age: ")
pause Wait for Keypress pause
os(command) Execute Shell Command os("echo Hi")

Comments

Use comments to document your code. Cave supports single-line and multi-line comments.

Syntax Type Usage
|| Single-line || This is a comment
|\ ... \| Block (Start/End) |\ This is a
multi-line comment \|
