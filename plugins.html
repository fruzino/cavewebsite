How to Build a Cave Plugin

Plugins are external programs that add new powers to the Cave programming language. Because Cave uses a simple communication protocol, you can write plugins in almost any language, including Python, C++, or even Batch!

The Concept

When you use #insert my_plugin, Cave starts your plugin as a background process. When you call a method like my_plugin.do_something(123), Cave sends a text message to your plugin's "Standard Input" (stdin). Your plugin reads the message, does its work, and can print results back to the screen.

Step 1: Create the Folder

All plugins live in the plugins/ folder of your Cave installation. Create a new folder there named after your plugin.

c:\Cave\plugins\
└── math_wizard\    <-- Your plugin folder

Step 2: The Launcher (run.bat)

Cave needs to know how to start your plugin. You must create a file named run.bat inside your plugin folder. This file tells Cave exactly what command to run.

Example run.bat:

@echo off
:: %%~dp0 ensures the script finds main.py in the same folder
python "%~dp0main.py"

Step 3: The Logic (main.py)

This is where the magic happens. While you can write this manually, Cave provides a plugin.py file to make plugins extremely simple.

Using the plugin.py

  1. Import the helper.
  2. Create a plugin instance.
  3. Use @plugin.method("name") to define what happens when Cave calls a function.

Example main.py:

import sys
import os

# Link to the Cave helper
sys.path.append(os.path.abspath(os.path.join(os.path.dirname(__file__), '..')))
from plugin import CavePlugin

plugin = CavePlugin()

# Use *args to safely handle inputs from Cave
@plugin.method("square")
def calculate_square(*args):
    if not args:
        return
    
    val = int(args[0])
    result = val * val
    print(f"The square of {val} is {result}")

if __name__ == "__main__":
    plugin.run()

Step 4: Testing your Plugin

Now, create a Cave script (test.cv) and try it out!

#insert math_wizard

write("Calling math wizard...")
math_wizard.square(5)

#cancel math_wizard

Run it with cave test.cv and you should see:
The square of 5 is 25

You can also check if a plugin is currently successfully loaded by using #inserted?("plugin_name") which evaluates to 1 or 0.

Advanced Header: Multiple Arguments

Cave supports passing multiple arguments to your plugin. For example, my_plugin.rect(10, 20). Your Python function will receive these as separate items in the args list: 

@plugin.method("rect")
def area(*args):
    if len(args) < 2:
        return
        
    w = int(args[0])
    h = int(args[1])
    print(f"Area: {w * h}")
